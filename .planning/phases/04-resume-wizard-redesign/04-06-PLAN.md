## Desktop Verification (Viewport >= 1024px)

### 1. Enhanced Vertical Sidebar Navigation

**Navigate to:** http://localhost:8080/resume-wizard/personal

**Check completion icons:**
- [ ] Empty steps show hollow circle icon (Circle from lucide-react)
- [ ] Steps with partial data show warning icon (AlertCircle, yellow/orange color)
- [ ] Completed steps show checkmark icon (CheckCircle2, green color)
- [ ] Icons update in real-time as you fill in form fields

**Check ATS weight badges:**
- [ ] High-impact steps (ATS weight >= 20%) show badge (e.g., "20%" in accent color)
- [ ] Badge appears next to step label (e.g., "Experience 20%")
- [ ] Tooltip on badge hover explains ATS impact: "High-impact section for ATS scoring"
- [ ] Steps: Experience (20%), Summary (15%), Skills (15%) show badges

**Check mini progress bars:**
- [ ] Each step shows subtle progress bar below label
- [ ] Progress bar fills based on field completion (0-100%)
- [ ] Progress updates in real-time as form fields are filled
- [ ] Empty step = 0% fill, partially filled = 30-70%, complete = 100%

**Check non-linear navigation:**
- [ ] Click any step in sidebar → navigates to that step (not blocked)
- [ ] Click incomplete required step → shows toast warning: "⚠️ [Step Name] is incomplete but you can continue"
- [ ] Click complete step → navigates without warning
- [ ] URL updates to match selected step (e.g., /resume-wizard/experience)
- [ ] Browser back/forward buttons work correctly with wizard steps

**Visual polish:**
- [ ] Active step has distinct highlight (background color, border, or shadow)
- [ ] Hover state on non-active steps (subtle background change)
- [ ] Smooth transitions when switching steps (no janky layout shifts)
- [ ] Icons and text aligned consistently (no misalignment issues)
